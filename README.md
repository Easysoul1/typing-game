# Typing Game

This is a simple typing game built with HTML, CSS, and JavaScript. The goal of the game is to type as many words as you can before the time runs out.

## How to Play

1.  Open the `solution/index.html` file in your web browser.
2.  Click the "Start" button to begin the game.
3.  A random word will be displayed on the screen.
4.  Type the word in the input field and press Enter.
5.  If you type the word correctly, your score will increase, and a new word will be displayed.
6.  The game ends when the timer reaches 0. Your final score will be displayed.

## Project Structure

*   `solution/index.html`: The main HTML file that creates the structure of the game.
*   `solution/index.css`: The CSS file that styles the game.
*   `solution/index.js`: The JavaScript file that contains the game logic.
